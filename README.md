# ticket by sport

### 首先安裝 node js

安裝相關依賴
`npm i`

取得 ASP.NET_SessionId
<img width="964" alt="getPrime website photo" src="https://github.com/wenbatman33/sport/blob/master/sid.png">

### config.toml 
填入相關參數
`sid= ASP.NET_SessionId  
place = 場次 
gameTime= 時間 
gameDate =日期
`
### 執行p12.js
`node tp12.js`

### 等候12點的來到
